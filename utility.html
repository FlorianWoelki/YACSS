# Utility

In YACSS you can find a lot of utility classes to customize the experience for the users on any webpage. Below are listed all utility classes you can use in any element.

Text

Font Sizes: 12, 14, 16, 18, 20, 24, 30, 36, 48, 60, 72, 96, 144, 288

class description
text-center Centers the text
uppercase Transform the text to uppercase letters
lowercase Transform the text to lowercase letters
font-thin Makes the font for the text thin
font-light Makes the font for the text light
font-normal Makes the font for the text normal
font-semibold Makes the font for the text semibold
font-bold Makes the font for the text bold
font-{12 -> 288} Set the size of the font to px (see font sizes)
line-height-{12 -> 288} Set the line-height of the font to px (see font sizes)

Spacing

Spacing Sizes: 0, 0-25, 0-5, 0-75, 1, 1-5, 2, 3, 4, 6, 8, 12, 16, 24, 32, 40, 48

class description
m-auto Margin auto on all sides
mt-auto Margin auto on top
mb-auto Margin auto on bottom
ml-auto Margin auto on left
mr-auto Margin auto on right
m-{0 -> 48} Sets margin on all sides to specific em (16 * x, see spacing sizes)
ml-{0 -> 48} Sets margin left to specific em (16 * x, see spacing sizes)
mr-{0 -> 48} Sets margin right to specific em (16 * x, see spacing sizes)
mt-{0 -> 48} Sets margin top to specific em (16 * x, see spacing sizes)
mb-{0 -> 48} Sets margin bottom to specific em (16 * x, see spacing sizes)
p-{0 -> 48} Sets padding on all sides to specific em (16 * x, see spacing sizes)
pl-{0 -> 48} Sets padding left to specific em (16 * x, see spacing sizes)
pr-{0 -> 48} Sets padding right to specific em (16 * x, see spacing sizes)
pt-{0 -> 48} Sets padding top to specific em (16 * x, see spacing sizes)
pb-{0 -> 48} Sets padding bottom to specific em (16 * x, see spacing sizes)

Shadow

Shadow Sizes: none, xsmall, small, medium, large, xlarge

class description
shadow-{none -> xlarge} Sets the shadow to specific size (see shadow sizes)

Border

Border Sizes: 0, 1, 2, 4, 8

class description
b-{0 -> 8} Sets white border (see border sizes)
bt-{0 -> 8} Sets white border top (see border sizes)
bb-{0 -> 8} Sets white border bottom (see border sizes)
bl-{0 -> 8} Sets white border left (see border sizes)
br-{0 -> 8} Sets white border right (see border sizes)

Border Radius

Border Radius Sizes: none, sm, reg, md, lg

class description
b-radius-{none -> lg} Sets border radius to specific size (see border radius sizes)
btl-radius-{none -> lg} Sets border top left radius to specific size (see border radius sizes)
btr-radius-{none -> lg} Sets border top right radius to specific size (see border radius sizes)
bbl-radius-{none -> lg} Sets border bottom left radius to specific size (see border radius sizes)
bbr-radius-{none -> lg} Sets border bottom right radius to specific size (see border radius sizes)

Other classes

class description
container Creates a responsive container for the element
pointer Sets the cursor to pointer
center Centers the element horizontally
circle Create a circular form for the element
pilled Create a pilled form for the element
block Set the display to block
inline-block Set the display to inline-block
pull-right Pull the element to the right
