# Buttons

By default buttons have the color of the specified accent color. It has accent-color-600 for the background and for the text color it has white or accent-color-600.

Button Types

There are different button types for styling your Webapp. We have primary buttons for displaying important action buttons and secondary buttons for displaying minor actions. Furthermore, we have tertiary buttons for link buttons.

Button Styles

You can use our background and color utility classes to easily customize your buttons.
