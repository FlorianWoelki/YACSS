# 12 grid layout

You can use the grid layout by defining a row and in that parent you can define columns from 1-12 by using for example col-4.

grid layout

Furthermore, you can use different grid classes to control the flow of the grid (e.g. on smaller devices), by using .sm-pos-x, .md-pos-x, .lg-pos-x to control the position on small, medium or large devices.
